# Documentation Assets

This directory contains documentation assets for the Rule Builder component.

## Screenshots

To add screenshots of the Rule Builder components:

1. Run the demo application: `npm run demo`
2. Take screenshots of the components in action
3. Save them in this directory with the following names:
   - `rule-builder-preview.png` - Main rule builder interface
   - `save-load-manager-preview.png` - Save/Load manager interface
4. The README.md will automatically display the images

## Recommended Screenshot Content

### Main Rule Builder (`rule-builder-preview.png`)
The screenshot should show:
- A rule with multiple conditions
- Nested rule groups with different combinators
- Various field types (text, number, select, boolean)
- The save/load functionality
- Output formats section
- Clean, professional appearance

### Save/Load Manager (`save-load-manager-preview.png`)
The screenshot should show:
- Rule Library interface with folder structure
- Search functionality in action
- Multiple saved rules organized in folders
- Toolbar with all action buttons
- Save rule dialog (if possible)
- Storage information display

## Image Specifications

- **Format**: PNG (preferred) or JPG
- **Size**: 1200px wide (for good GitHub display)
- **Quality**: High resolution for clarity
- **Background**: Light theme for better visibility
